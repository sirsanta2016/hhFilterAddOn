# 🚀 HH Filter AddOn – One-Click Filters for HellHades Optimiser

Cuts artifact filtering from 30 clicks to 1.

Enhance your experience on the HellHades Raid Optimiser with fast, reliable, one-click filtering and quick champion rating visibility.

This userscript reduces repetitive actions, improves artifact and accessory filtering, and retrieves the overall champion rating from HellHades.

---

## ✅ Features

- One-click preset filters (Dragon, Fire Knight, Ice Golem, etc.)
- Quick faction selection
- Automated accessory filtering (Ring, Amulet, Banner)
- Instant filter reset
- Retrieves and displays the overall champion rating from HellHades
- Works with dynamic dropdowns and overlays

---

## ⚡ Benefits

- Reduces 20–30 manual clicks to a single action  
- Saves time during gear optimization  
- Improves consistency and usability  
- Helps you quickly see champion rating context while optimizing builds  

---

## 🔒 Safety & Compatibility

- Runs only on: https://raidoptimiser.hellhades.com/  
- Uses HellHades-related champion data only for display purposes  
- No personal data collection  
- No ads or tracking  
- Open source

## 📦 Installation

👉 Install directly from Greasy Fork:  
https://greasyfork.org/en/scripts/573895-hh-raid-optimiser-one-click-filters-presets-factions  

Or:

1. Install a userscript manager (e.g., Tampermonkey)  
2. Install the script from Greasy Fork  
3. Open HellHades Optimiser and use the new controls
